WEB vs INTERNET
A internet é uma rede global de conexão entre computadores, utilizando-se de tecnologias desde fios de cobra até satélites para obter as melhores conexões entre as máquinas ao redor do mundo. Dispõe-se de protocolos como o endereço IP para identificar os dispositivos conectados. A internet oferece diversos serviços, como e-mails, upload e download de arquivos, e entre outros.
Já a WEB atua dentro da internet, como um serviço, a WEB (também chamada de World Wide Web) funciona baseado em páginas com linguagem de marcação de textos (Hypertext Markup Language), desde a sua criação, a WEB trabalha com protocolos criados por Tim Berners-Lee, que propôs o uso de requisições HTTP.
